# Internal Docs

Date: 2026-03-07

and audits, but is not the main source of truth for platform architecture.

Typical contents:

- review trackers
- audit logs
- raw inventories
- temporary documentation generated during cleanup or verification work
- narrow engineering notes
- historical implementation contracts

- `docs/assistant-agent-RAG-overview.md`
- `docs/chat-page-system-map.md`
- `docs/RAG_SETUP.md`
